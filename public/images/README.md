# 写真の入れ方（差し替えガイド）

このフォルダに写真を置くと、トップページの画像が本物の写真に変わります。

## 手順
1. 使いたい写真をこのフォルダ（`public/images/`）に入れる。
   例: `feature.jpg`, `f1.jpg`, `s1.jpg` など。
2. `lib/data.ts` を開き、対応する項目に `image` を追記する。
   例:
   ```ts
   {
     id: "a1",
     title: "AIで楽天ビューティー閲覧数1位になった話",
     // ...
     image: "/images/feature.jpg",   // ← この1行を足すだけ
   }
   ```
3. 保存すると、その場所が写真に切り替わります（`image` が無い項目は
   これまで通りのデザイン入りプレースホルダのまま）。

※ 写真は著作権・利用規約に注意して用意してください
  （例: Unsplash / Pexels などの無料素材、または自店で撮影したもの）。
